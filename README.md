# CUSTOMER_BEHAVIOR_ANALYSIS

Projeto de Análise de Dados Comportamentais de Clientes, desenvolvido com Python, SQL e Power BI, cobrindo todo o pipeline de dados — desde a coleta, tratamento e modelagem até a análise exploratória e o desenvolvimento de dashboards interativos.
Adicionalmente, foram aplicadas técnicas de Inteligência Artificial como suporte na estruturação do dashboard e na elaboração da apresentação do relatório, contribuindo para uma comunicação mais eficaz dos insights e maior assertividade na tomada de decisão.

Python
Utilizado para:
Limpeza e tratamento de dados
Análise exploratória (EDA)
Manipulação de grandes volumes de dados com bibliotecas como Pandas e NumPy

SQL
Aplicado para:

Extração de dados a partir de bases relacionais
Criação de consultas otimizadas para análise
Organização e transformação dos dados em etapas iniciais

Power BI
Responsável por:

Criação de dashboards interativos
Visualização de indicadores-chave (KPIs)
Apresentação dos insights de forma clara e dinâmica

Inteligência Artificial
Utilizada como suporte para:

Auxiliar na estruturação e design do dashboard
Apoiar a elaboração e organização da apresentação do relatório
Melhorar a clareza na comunicação dos resultados

Resultados Esperados
Identificação de padrões de comportamento dos clientes
Geração de insights estratégicos
Apoio à tomada de decisão baseada em dados
Comunicação eficiente através de visualizações interativas
